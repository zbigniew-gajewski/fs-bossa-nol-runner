Nol Runner

Projekt fs-bossa-nol-runner służy do automatycznego uruchamiania aplikacji Notowania Online 3 (Nol 3) z poziomu przeglądarki Internet Explorer. Projekt wykonuje automatycznie sekwencję czynności:

  • uruchomienie przeglądarki (Internet Explorer 11)
  • logowanie do konta w bossa.pl
  • uruchomienie aplikcaji Nol 3
  • zamknięcie przeglądarki

Projekt wykorzystuje:

  • Canopy (F# API do Selenium)
  • Argu (F# API do parsowania argumentów)

Wymagania

Aplikacja była przygotowywana i testowana w nastęującym środowisku:

  • Windows 10 (1703)
  • Internet Explorer 11
  • Nol 3 (v. 3.1.15.142.I.7)
  • Visual Studio 2017 Community (v. 15.5.2) z komponentami F# desktop i Git

Driver do przeglądarki

Canopy do działania potrzebuje drivera Selenium (IEDriverServer.exe). Po pobraniu i po rozpakowaniu plik IESeleniumDriver.exe należy wgrać bezpośrednio na dysk 'C'.

Projekt był testowany z driverem IEDriverServer_x64_3.8.0.zip. Inne wersje drivera (v. 3.8) można pobrać z: IEDriverServer_3.8.0.

Uruchomienie

Po skompilowaniu aplikacji, binaria można przenieść do dowolnego katalogu:


Aplikację FsBossaNolRunner.exe należy uruchomić z parametrem:

FsBossaNolRunner.exe --credentials [login] [hasło]


Jeśli parametr nie zostanie podany, to login i hasło zostaną pobrane ze zmiennej środowiskowej bossaCredentials. Jeśli parametrów nie ma też w zmiennej środowiskowej, aplikacja nie uruchomi ani przeglądarki ani Nol 3.
